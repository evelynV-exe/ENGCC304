## โจทย์
จงเขียนโปรแกรมภาษา C ที่มีโครงสร้างครบถ้วน โดยประกอบไปด้วยการ include และ main function (ชนิดข้อมูล int) เพื่อแสดงข้อความ `"Hello World."` บนหน้าจอ

## FIX CODE
```c++
#include <stdio.h>

int main() {
    printf("Hello World.");
    return 0;
}
```
